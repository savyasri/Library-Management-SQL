# Digital Library Management System (SQL)

This project is a simple SQL-based library system.

## Features:
- Store books information
- Store users
- Track issued books
- Use JOIN queries to combine data

## Tables:
- books
- users
- issued_books

## Queries:
- View all books and users
- JOIN to see issued books
- Count total issued books
- Filter low stock books

## Tools Used:
- MySQL Workbench
